# 📝 درباره پروژه

این پروژه یک نرم‌افزار دسکتاپ مدرن و کارآمد برای **زمان‌بندی، مدیریت و یادآوری کلاس‌های درسی و آموزشی** است که با زبان **Python** و رابط کاربری گرافیکی **PyQt6** توسعه یافته است. هدف این نرم‌افزار، نظم‌بخشی به ساعات یادگیری دانشجویان، دانش‌آموزان و مدرسین است تا هیچ کلاسی را فراموش نکنند.

---

# ⚡ ویژگی‌های کلیدی نرم‌افزار

* **رابط کاربری مدرن (UI/UX):** استفاده از تم زیبای Fusion، فونت‌های بهینه‌شده و چیدمان کاملاً راست‌چین و بومی‌سازی شده.
* **پایگاه داده داخلی (SQLite):** ذخیره‌سازی کاملاً خودکار و امن اطلاعات بدون نیاز به پیکربندی‌های پیچیده.
* **سیستم یادآوری هوشمند (Notification):** ارسال اعلان‌های آنی سیستم‌عامل ۱۰ دقیقه قبل یا در زمان شروع کلاس بعدی با کمک کتابخانه `plyer`.
* **مدیریت هوشمند زمان:** هماهنگ‌سازی پیشرفته روزهای هفته (فارسی) با تاریخ سیستم و مدیریت فایل تنظیمات JSON.
* **طراحی واکنش‌گرا (Responsive Table):** تغییر خودکار سایز فونت‌ها و ردیف‌های جدول متناسب با بزرگ یا کوچک کردن پنجره نرم‌افزار.

---

# 🛠️ نحوه کار با نرم‌افزار و معماری کد

کدهای این پروژه به صورت کاملاً ماژولار و بهینه نوشته شده‌اند:
1. **شروع برنامه:** در اولین اجرا، یک پنجره (DayConfigDialog) باز می‌شود تا روز فعلی هفته را برای هماهنگی دقیق‌تر با سیستم مشخص کنید؛ این تنظیمات در فایل `config.json` ذخیره می‌شود.
2. **جدول زمان‌بندی:** یک جدول جامع از ساعت ۹ صبح تا ۲۳ شب و روزهای شنبه تا جمعه در اختیار شماست. کافیست روی هر خانه دابل کلیک کرده و نام کلاس یا دوره آموزشی خود را بنویسید.
3. **ذخیره‌سازی دیتابیس:** با کلیک روی دکمه **"ذخیره تغییرات"**، توابع `save_schedule_to_db` فعال شده و اطلاعات را در فایل `class_schedule.db` ثبت می‌کنند.
4. **بررسی کلاس بعدی:** برنامه هر ۶۰ ثانیه یک‌بار (با استفاده از سیستم `Timer`) زمان را چک کرده و نام کلاس بعدی را در بالای پنجره به رنگ قرمز نمایش می‌دهد و همزمان نوتیفیکیشن ویندوز/لینوکس را ارسال می‌کند.
