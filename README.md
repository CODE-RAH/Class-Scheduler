# 📝 درباره پروژه

این پروژه یک نرم‌افزار دسکتاپ مدرن و کارآمد برای **زمان‌بندی، مدیریت و یادآوری کلاس‌های درسی و آموزشی** است که با زبان **Python** و رابط کاربری گرافیکی **PyQt6** توسعه یافته است.

هدف این نرم‌افزار، نظم‌بخشی به ساعات یادگیری دانشجویان، دانش‌آموزان و مدرسین است تا هیچ کلاسی را فراموش نکنند.

---

# ⚡ ویژگی‌های کلیدی نرم‌افزار

- **رابط کاربری مدرن (UI/UX)**  
  استفاده از تم زیبای Fusion، فونت‌های بهینه‌شده و چیدمان کاملاً راست‌چین و بومی‌سازی شده.

- **پایگاه داده داخلی (SQLite)**  
  ذخیره‌سازی کاملاً خودکار و امن اطلاعات بدون نیاز به پیکربندی‌های پیچیده.

- **سیستم یادآوری هوشمند (Notification)**  
  ارسال اعلان‌های آنی سیستم‌عامل ۱۰ دقیقه قبل یا در زمان شروع کلاس بعدی با کمک کتابخانه `plyer`.

- **مدیریت هوشمند زمان**  
  هماهنگ‌سازی پیشرفته روزهای هفته (فارسی) با تاریخ سیستم و مدیریت فایل تنظیمات JSON.

- **طراحی واکنش‌گرا (Responsive Table)**  
  تغییر خودکار سایز فونت‌ها و ردیف‌های جدول متناسب با بزرگ یا کوچک کردن پنجره نرم‌افزار.

---

# 🛠️ نحوه کار با نرم‌افزار و معماری کد

کدهای این پروژه به صورت کاملاً ماژولار و بهینه نوشته شده‌اند:

1. **شروع برنامه**  
   در اولین اجرا، یک پنجره (`DayConfigDialog`) باز می‌شود تا روز فعلی هفته را برای هماهنگی دقیق‌تر با سیستم مشخص کنید؛ این تنظیمات در فایل `config.json` ذخیره می‌شود.

2. **جدول زمان‌بندی**  
   یک جدول جامع از ساعت ۹ صبح تا ۲۳ شب و روزهای شنبه تا جمعه در اختیار شماست. کافیست روی هر خانه دابل کلیک کرده و نام کلاس یا دوره آموزشی خود را بنویسید.

3. **ذخیره‌سازی دیتابیس**  
   با کلیک روی دکمه **«ذخیره تغییرات»**، توابع `save_schedule_to_db` فعال شده و اطلاعات را در فایل `class_schedule.db` ثبت می‌کنند.

4. **بررسی کلاس بعدی**  
   برنامه هر ۶۰ ثانیه یک‌بار (با استفاده از سیستم `Timer`) زمان را چک کرده و نام کلاس بعدی را در بالای پنجره به رنگ قرمز نمایش می‌دهد و همزمان نوتیفیکیشن ویندوز/لینوکس را ارسال می‌کند.

# 🚀 راهنمای نصب و اجرای پروژه (ویژه مبتدیان)

اگر تا به حال با برنامه‌های پایتون کار نکرده‌اید، با انجام مراحل زیر می‌توانید این نرم‌افزار را روی کامپیوتر خود اجرا کنید.

---

## 📥 مرحله اول: نصب پایتون (Python)

این برنامه با زبان پایتون نوشته شده است؛ بنابراین ابتدا باید Python را روی سیستم خود نصب کنید.

1. وارد سایت رسمی Python شوید و آخرین نسخه را دانلود کنید.
2. هنگام نصب، حتماً گزینه زیر را فعال نمایید:

```text
Add Python to PATH
```

3. سپس روی دکمه **Install Now** کلیک کنید تا نصب کامل شود.

---

## 📦 مرحله دوم: دانلود فایل‌های پروژه

1. از بالای صفحه گیت‌هاب، روی دکمه سبز رنگ **Code** کلیک کنید.
2. گزینه **Download ZIP** را انتخاب نمایید.
3. فایل دانلود شده را از حالت فشرده خارج کنید.

---

## ⚙️ مرحله سوم: نصب پیش‌نیازها

1. وارد پوشه پروژه شوید (جایی که فایل `01.py` قرار دارد).
2. داخل پوشه، کلید `Shift` را نگه دارید و راست کلیک کنید.
3. گزینه **Open PowerShell window here** یا **Open in Terminal** را انتخاب نمایید.

سپس دستور زیر را اجرا کنید:

```bash
pip install pyqt6 plyer
```
